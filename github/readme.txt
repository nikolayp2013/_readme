github - это репозиторий для проектов

чтобы заработала cmd после установки git нужна перезагрузка компьютера

после установки Git-2.21.0-64-bit неоходимо сконфигурировать:
для глобальных настроек
git config --global user.name "Nikolay Perov"
git config --global user.email "nikolayp2013@gmail.com"

для клонирования репозитория необходимо:
выбрать каталог куда хотим клонировать
git clone https://github.com/nikolayp2013/my-project.git
